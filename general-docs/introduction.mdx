Identety is an open-source identity provider implementing OAuth 2.0 and OpenID Connect standards. It offers secure and flexible identity and access management with OAuth2 flows, OIDC claims/endpoints, support for public, private, and machine clients, and granular role-based access control.

Identety is available as a single-tenant self-hosted solution or as a multi-tenant cloud SaaS at identety.dev. The self-hosted version uses API keys for protection, while the cloud version uses cloud account management.

Built on modern technologies like NestJS and PostgreSQL, with API docs powered by OpenAPI. Identety aims to democratize identity management by providing an easy-to-use yet robust solution that adheres to industry standards.

## Mission

To make powerful identity and access management accessible to developers and organizations of all sizes through open-source software and SaaS.

## Vision

Become the go-to open-source identity platform by combining exceptional developer experience with enterprise-grade security and flexibility.

Whether self-hosting or using the cloud offering, Identety empowers teams to integrate secure authentication and authorization into their applications with ease.
